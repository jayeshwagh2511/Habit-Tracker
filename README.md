# Habit Tracker

A Java + MySQL based CLI application to track daily habits and generate simple AI-style productivity suggestions.

## Features
- Add and manage habits
- Log daily hours
- View records with joined data
- AI-based suggestions

## Tech Stack
- Java (JDBC)
- MySQL

## Project Structure
Main.java  
DBConnection.java  
database.sql  

## How to Run
1. Setup MySQL using database.sql
2. Update DBConnection.java with your username & password
3. Compile:
   javac Main.java DBConnection.java
4. Run:
   java Main
